VULCAN STOCK TRACKER — IMAGE FILE GUIDE
========================================

The app references images by fixed filenames. Place your PNG files
in the correct folders and they will appear automatically in the UI.
If an image is missing the app will still work — it shows an icon placeholder.

TIPS
────
• PNG files with transparent backgrounds work best on the dark theme.
• Recommended size: 256×256 px (displayed at 64×64 for products, 28×28 for parts).
• File names are case-sensitive on Linux/Render — use exactly the names above.
• After adding images, no code changes are needed; just refresh the browser.
